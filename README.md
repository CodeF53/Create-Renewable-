# CreatePlus
A Minecraft Datapack / QuiltMod for 1.18.2.

Base compatibility enchantments for Create Fabric.

Supported Mods:
- Tech Reborn
- Farmer's Delight
- Alloy Forgery
- Applied Energistics 2
- Betternether
- Immersive Weathering
- More Ores
- Promenade
- Unearthed
- Universal Ores
- Alloygery
